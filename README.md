# Xclusiv Footwear — MERN Stack

## Project Structure
```
xclusiv-footwear/
├── frontend/   → React + Vite + Tailwind CSS
└── backend/    → Node.js + Express + MongoDB Atlas
```

## Quick Start

### 1. Backend
```bash
cd backend
npm install
# Create .env file (see .env.example)
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

### backend/.env
```
MONGO_URI=mongodb+srv://<user>:<password>@cluster.mongodb.net/xclusiv
PORT=5000
WHATSAPP_NUMBER=919XXXXXXXXX
```

### frontend/.env
```
VITE_API_URL=http://localhost:5000/api
VITE_WHATSAPP_NUMBER=919XXXXXXXXX
```
